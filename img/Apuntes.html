    Un documento html 5 empieza con la linea "!DOCTYPE html"
    Con el editor Visual Studio Code simplemente añadimos la ! y al dar intro nos aparece toda parte inicial, incluido el
     "head"
                ---< !DOCTYPE html >
                < html lang="es" >
                < head >
                    < meta charset="UTF-8" >
                    < meta http-equiv="X-UA-Compatible" content="IE=edge" >
                    < meta name="viewport" content="width=device-width, initial-scale=1.0" >
                    < title >Apuntes< /title >
                < /head >---

    html lang="es" indica que la página está en español. Si lo dejamos por defecto "en" veremos que al entrar con el navegador
    nos pregunta si queremos traducir, ya que el idioma del navegador no es el mism.
    Así que yo lo pondré en español

    Dentro de head tenemos las etiquetas meta que nos dan formatos importantes para el navegador.
                charser = "UTF-8" es el código que necesita para los carácteres especiales, por ejemplo para los acentos
                abiertos $ &

    La estructura es muy importante para el tema de SEO
    
                html 
                 head 
                  body
                    header 
                      nav
                    main 
                        section 
                        aside
                        nav 
                    footer
    

    Esta sería una estructura básica.

    Todo lo que se ve estará dentro del body 
    Todo lo que no se ve, es decir, los formatos, van dentro del head 

    Los formatos pueden incluirse también dentro del body, pero esto no va bien para SEO. 
    Los formatos dentro de head van en CSS 

    El header es la cabecera de la página. Por lo general incluye un menú, así que le añadiremos también un nav 

    El main es la parte central de la página. Dendro iremos añadiendo todas las partes de nuestra web.

    El aside es la zona tando derecha como izquierda que contenga información y el section sería la parte central. También podemos tener más nav dentro de main.
    Todo esto irá dentro de main 
    Finalmente tenemos el footer, que es el pie de página.

    la zona en la que se encuentra.
