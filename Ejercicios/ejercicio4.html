Como crear una página web

Pasos a seguir

En este documento se detallan los elementos necesarios para la creación de una página web y algunos recordatorios:

Elementos necesarios:

  • Doctype
  • HTML
    • Head
      • Meta charset
      • Title
    • Body
      • Header
      • Nav
      • Main
      • Aside
      • Section
      • Article
      • Footer

Recordatorios:

  1. La etiqueta br no debe utilizarse para generar espacios
  2. Las etiquetas h1-h6 deben utilizarse en función de la relevancia de los titulares, no por sus estilos
  3. Toda etiqueta debe abrirse y cerrarse a excepción de algunas como meta, br o imbg
  4. Un enlace que desvie a otra pestaña deberá incluir el atributo target="_blank"
